# YoutubeGrabber - 網頁版操作指南 🌻

本專案為 **Youtube 擷取工具網頁版**，所有影片下載、剪輯與語音辨識運算皆在您的本機電腦上執行，提供極致美觀、自適應的網頁操作面板。

---

## 🛠️ 準備工作

### 1. 安裝環境依賴
確保您已經安裝了 Python 3.8+。在專案目錄下執行：
```bash
pip install -r requirements.txt
```

### 2. 核心組件檢測 (FFmpeg)
* 本專案需要 `ffmpeg.exe` 與 `ffprobe.exe` 才能進行下載與裁剪。
* **全自動下載**：當您首次開啟伺服器時，系統會自動在背景檢查本機是否存在這些組件。若缺失，後端會自動下載並部署在背景，無須手動干預。

---

## 🚀 如何啟動網頁版

您可以選擇 **直接執行原始碼** 或 **編譯成單一獨立執行檔** 來運行：

### 方式一：直接執行原始碼 (Python)
1. 切換至專案目錄。
2. 啟動伺服器：
   ```bash
   python src/main.py
   ```
3. 伺服器啟動後，會**自動為您開啟瀏覽器**導向網頁端操作面板！
   （您也可以手動訪問：[https://FirePhoenix1001.github.io/WedTest/](https://FirePhoenix1001.github.io/WedTest/)）

---

### 方式二：使用 PyInstaller 打包為獨立執行檔 (EXE)
本專案提供隨攜式單一執行檔配置，您可以直接將後端打包為一個無 Console 黑色視窗的獨立綠色軟體：

1. 安裝 `pyinstaller`：
   ```bash
   pip install pyinstaller
   ```
2. 執行打包指令：
   ```bash
   pyinstaller main.spec
   ```
3. 打包完成後，會在 `dist/` 資料夾下生成 `SunflowerWebStudio.exe`。
4. **隨攜式執行**：您可以將這個 `SunflowerWebStudio.exe` 拷貝到任何地方（例如：**桌面**）直接雙擊執行！
   * 執行後，它會自動開啟瀏覽器導向網頁端。
   * **所有下載的影片、剪輯與辨識結果，都會直接出現在該執行檔所在的資料夾下**。

---

## 🌟 功能說明

1. **📥 下載 YouTube**：貼上影片網址，選擇下載模式（僅影片/僅聲音/影音合併），點擊下載，可在下方實時觀看日誌與進度。每次啟動時後端會自動背景更新 `yt-dlp` 以免 403 報錯。
2. **✂️ 本地剪輯**：輸入本地檔案絕對路徑，填入開始與結束時間，即可利用 FFmpeg 無損複製技術在數秒內極速裁剪。
3. **🎤 語音辨識**：選擇本地音訊/影片檔案路徑，套用不同大小的 Whisper 模型，一鍵產出帶時間戳記的繁體中文字幕 `.txt` 檔。
4. **📂 檔案管理**：讀取執行檔當前目錄下的所有媒體與辨識文字，提供「電腦開啟」（呼叫系統播放器）、「剪輯」、「辨識」與「刪除」的快速鍵整合。
5. **⚙️ 移除組件**：可在左下角一鍵刪除電腦本機的 `ffmpeg` 與 `ffprobe` 檔案，恢復初始狀態。
