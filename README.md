# TextPresent | 文字美學展示工坊 🎨

一個專為呈現「任何文字」而設計的極致美學展示工坊。您可以利用本工具輸入任何金句、詩詞、代碼或名言，並透過自訂字型、版面配置、動態特效及多款精心調配的主題，即時呈現最具藝術感的排版效果，還支援一鍵「全螢幕展示」與「導出高品質 PNG 卡片」！

## ✨ 特色功能

1. **豐富的主題風格**：
   - 🌌 **極光玻璃**：毛玻璃磨砂質感，背後有神秘幽微的極光流動。
   - 🌅 **暖陽落日**：粉色與黃色溫暖交織，適合現代感性的金句。
   - 🔮 **霓虹幻夜**：賽博龐克風格的霓虹邊框與文字發光特效。
   - ❄️ **北歐晨曦**：極簡、寧靜且透亮的藍灰色調，帶有純粹的空氣感。
   - 📰 **紙質社論**：重現經典報紙的黑白排版與極高對比度。
   - 🟩 **矩陣駭客**：黑客綠色等寬代碼主題，極客感十足。
2. **多樣化的排版調整**：
   - 實時調整字型（Outfit、Playfair Display、思源宋體、Fira Code 等）。
   - 滑桿調整字型大小、行高、字距。
   - 三種對齊方式（靠左、置中、靠右）。
3. **靈感快選**：
   - 內建四類經典文字（經典金句、優美詩詞、質感代碼、極簡隨筆），點擊即可一鍵套用完美配置。
4. **極致動態特效**：
   - 多款文字載入動畫（淡入漸顯、滑入展開、焦點放大、迷霧漫延）。
   - 動態流光背景粒子，讓網頁栩栩如生。
5. **展示與導出**：
   - **全螢幕模式**：隱藏一切控制器，只留下精美的文字展示卡片與優美背景，適合簡報與投影。
   - **卡片導出**：使用 `html2canvas` 技術，一鍵下載 3 倍超高解析度的 PNG 圖片，方便分享至社群平台！

## 🛠️ 如何在本地執行

因為使用了靜態 HTML5 網頁技術，您可以透過任何本地 Web 伺服器來預覽這個專案。

### 方法一：使用 Python 快速開啟 (推薦)
如果您有安裝 Python，可在專案目錄下執行以下指令：
```bash
python -m http.server 8000
```
然後在瀏覽器中打開 `http://localhost:8000`。

### 方法二：直接點擊
在 Windows 檔案總管中，直接雙擊 `index.html` 即可在瀏覽器中開啟！

## 🚀 如何部署至 GitHub Pages (`github.io`)

這個專案是 100% 純前端靜態專案，非常適合免費部署至 GitHub Pages：

1. **建立專案庫 (Repository)**：
   - 在 GitHub 上建立一個新的公開專案庫，例如命名為 `text-present`。
2. **上傳程式碼**：
   - 將本目錄下的 `index.html`、`style.css`、`app.js` 上傳（或 push）到該專案庫的 `main` 分支。
3. **啟用 GitHub Pages**：
   - 進入該專案庫的 **Settings** -> **Pages**。
   - 在 **Build and deployment** 下的 **Source** 選擇 `Deploy from a branch`。
   - **Branch** 選擇 `main`，路徑選擇 `/ (root)`，並點擊 **Save**。
4. **預覽您的網站**：
   - 等待幾分鐘後，您的範例網站將會發布在 `https://<您的帳號>.github.io/text-present/`！
