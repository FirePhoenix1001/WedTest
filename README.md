# YoutubeGrabber - 網頁版操作指南 🌻

本專案為 **Youtube 擷取工具網頁版**，所有影片下載、剪輯與語音辨識運算皆在您的本機電腦上執行，提供極致美觀、自適應的網頁操作面板。

## 🛠️ 準備工作

1. **安裝 Python 套件**：
   確保您已經安裝了所需的依賴與網頁伺服器套件。在專案目錄下執行：
   ```bash
   pip install -r requirements.txt
   pip install flask flask-cors
   ```

2. **核心組件檢測 (FFmpeg)**：
   * 本專案需要 `ffmpeg.exe` 與 `ffprobe.exe` 才能進行下載與裁剪。
   * **免手動下載**：當您啟動伺服器並首次打開網頁時，系統會自動偵測您的電腦是否存有這些組件。若缺少，網頁上方將會顯示一個橘色警告橫幅，點選 **「一鍵自動安裝」** 即可全自動下載部署！

---

## 🚀 如何啟動網頁版

1. 在終端機 (Terminal / CMD / PowerShell) 中切換至專案目錄。
2. 啟動伺服器：
   ```bash
   python src/main.py
   ```
3. 啟動成功後，終端機會顯示：
   `[Sunflower] 向日葵本地網頁伺服器啟動中，請打開 http://localhost:8000`
4. 開啟您的網頁瀏覽器，造訪 [http://localhost:8000](http://localhost:8000) 即可開始使用！

---

## 🌟 功能說明

1. **📥 下載 YouTube**：貼上影片網址，選擇下載模式（僅影片/僅聲音/影音合併），點擊下載，可在下方實時觀看日誌與進度。
2. **✂️ 本地剪輯**：輸入本地檔案絕對路徑，填入開始與結束時間，即可利用 FFmpeg 無損複製技術在數秒內極速裁剪。
3. **🎤 語音辨識**：選擇本地音訊/影片檔案路徑，套用不同大小的 Whisper 模型，一鍵產出帶時間戳記的繁體中文字幕 `.txt` 檔。
4. **📂 檔案管理**：讀取專案目錄下的所有媒體與辨識文字，提供「電腦開啟」（呼叫系統播放器）、「剪輯」、「辨識」與「刪除」的快速鍵整合。
5. **⚙️ 移除組件**：可在左下角一鍵刪除電腦本機的 `ffmpeg` 與 `ffprobe` 檔案，恢復初始狀態。
