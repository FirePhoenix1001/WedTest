# YoutubeGrabber - 使用者操作說明書 🌻

歡迎使用 **Youtube 擷取與編輯工具網頁版**！本工具是一款精美且實用的綠色隨攜式軟體。所有影片下載、無損裁剪與 Whisper AI 語音辨識運算皆在您的本機電腦上安全執行，完全不依賴第三方雲端伺服器，保障您的隱私。

---

## 🚀 快速開始使用

本軟體採用 **「C# 啟動器外殼 (`SunflowerLauncher.exe`) + 背景 Python 伺服器」** 的架構，無須額外繁瑣的手動設定：

1. **解壓縮並放置**：將程式資料夾放置於本機磁碟的任何位置（建議路徑中不要包含特殊字元或權限受限的資料夾）。
2. **雙擊啟動**：雙擊專案目錄下的 **`SunflowerLauncher.exe`**。
   * **單一執行保證**：啟動器會自動清理先前殘留或重複開啟的程式，不用擔心重複點擊造成資源浪費。
   * **自動對接**：啟動器會默默在背景執行，並自動開啟您的瀏覽器導向操作網頁。
   * **可攜工作路徑**：所有下載與產出的檔案（影片、音訊、語音辨識產生的 `.txt` 文字檔）都會直接存放在 `SunflowerLauncher.exe` 所在的資料夾下，方便您隨時用隨身碟帶走。

## 💎 特色功能說明

### 1. 系統匣常駐與彻底退出 🌻
* 程式啟動後，會縮小至系統右下角工作列的隱藏圖示區。
* 右鍵點選向日葵圖示，即可呼叫選單：
  * **開啟向日葵網頁端**：手動在瀏覽器中開啟操作面板。
  * **顯示日誌控制台**：顯示一個精緻的暗黑日誌視窗，供您查看 Python 後端執行與下載日誌。
  * **結束並關閉伺服器**：安全退出，將會一併強制結束所有 Python 背景進程並釋放 Port 8000 連線埠。

### 2. 依賴組件「一鍵網頁下載」
* **無須手動設定環境**：如果程式檢測到您缺少視訊處理的核心組件（FFmpeg、FFprobe 或 yt-dlp），網頁頂部會彈出黃色的 **「偵測到缺少必要核心組件」** 提醒橫幅。
* **一鍵安裝**：您只需點擊橫幅右側的 **「一鍵自動安裝」**，網頁便會自動在背景下載並為您部署完畢。進度與安裝日誌都會實時在網頁下方的終端機及進度條顯示。

### 3. 未連線警告互斥提示
* 如果您忘記開啟 `SunflowerLauncher.exe` 或連線中斷，網頁頂部會顯現紅色的 **「未連線至背景伺服器」** 橫幅。
* 此時程式會自動隱藏缺少組件的黃色提示，以專注於提醒您建立連線。重新連線後，警示會自動消退。

### 4. 智慧本地檔案選取與拖放解析
* **本機選取檔案**：在本地視訊裁剪及 AI 語音辨識的輸入欄位旁，設有 **「瀏覽檔案」** 按鈕。點擊後會直接喚起本機的 **Windows 原生選取檔案視窗**，您無須再手動複製冗長的絕對路徑。
* **拖曳自動解析路徑**：您也可以直接將專案目錄下的影片拖曳到路徑輸入框中，網頁端會自動比對並對接填入該檔案在本機上的絕對路徑。

### 5. 一鍵清空環境
* 當您不再需要使用本程式，您可以點選左側側邊欄的 **「一鍵清空環境」** 按鈕。
* 程式將只會移除本機下載部署的 `tools/` 核心組件（不會影響您已下載的影片），讓本機電腦重回乾淨狀態。

## 📂 功能操作說明

### 📥 YouTube 下載功能
1. 複製 YouTube 影片的網址，並貼入輸入框中。
2. 選擇下載模式：
   * **僅影片**：擷取無聲的高清影片。
   * **僅聲音 (MP3)**：將影片音訊轉換為優質的 MP3 檔案。
   * **影音合併 (推薦)**：下載並封裝出最完美的 Full HD/4K 影音合併檔。
3. 點擊 **「開始下載」**，網頁下方的日誌與進度條將會呈現即時進度。

### ✂️ 本地媒體裁剪
1. 點選 **「瀏覽檔案」** 選取您電腦上的影片或音訊（或直接在下方檔案管理對想剪輯的檔案點擊「剪輯」）。
2. 在時間輸入框設定剪輯的「起始時間」與「結束時間」。
3. 點擊 **「開始無損剪輯」**，本程式將在數秒內無損導出裁剪後的影片，不損害任何畫質。

### 🎤 Whisper AI 語音辨識
1. 選取您要辨識的影音檔案。
2. 選擇 Whisper 模型大小（模型越小速度越快，模型越大精準度越高）。
3. 勾選是否包含時間戳記，點擊 **「開始辨識」**。
4. 辨識完成後，會在影片同目錄下自動儲存一份 `_辨識結果.txt` 的正體中文文字檔。

### 📁 本地檔案管理
* 網頁設有檔案總覽表格，您可以在網頁中直接看到所有已下載或裁剪的檔案。
* 點擊 **「電腦開啟」** 可直接使用電腦的預設播放器播放；亦支援點選 **「刪除」** 永久移除本機檔案。
